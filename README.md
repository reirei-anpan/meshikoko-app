### アプリ名<br>
- **メシココ**

### 概要<br>
- **メシココ** は、友人同士での飲食店選びを支援するWebアプリケーションです。<br>
場所、時間、食事の種類、金額、雰囲気など、飲食店選びに必要な複数の要素を考慮し、グループ全員の意見を集約して最適な選択を支援します。
このアプリは、飲食店の候補募集から最終的な予約確定までのプロセスを簡単かつ効率的に行うことを目的としています。
